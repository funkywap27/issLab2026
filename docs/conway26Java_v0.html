Introduction

Goal: Realizzazione in Java del GAME OF LIFE DI CONWAY limitatamente al requisito R1.

Requirements

R1  Realizzare una versione in Java del gioco Life di Conway, come gioco zero-player. 
    Il gioco consiste nell’introdurre una Griglia di Celle il cui stato (cella ‘viva’ o cella ‘morta’) 
    evolve come stabilito dallle regole di ConwayLife

R2  L’utente umano deve poter:
     - specificare la configurazione iniziale della griglia del gioco	
     - vedere l’evoluzione del gioco in forma opportuna 
        (si veda Problema della vista del gioco )
     - fermare e far ripartire l’evoluzione del gioco
     - pulire (a gioco fermo) la configurazione della griglia del gioco

Requirement analysis


    In discussione con il committente si è rilevato che la cella è un ente che modifica il proprio stato interno, che puo avere solo valore true o false.

    public interface ICell {
        void setStatus(boolean v); //deriva dal fatto che la cella è un ente e che puo avere la capacita di moodificare il prorpio stato interno in base a un valore booleano che puo essere o true o false
        boolean isAlive(); // una cella puo avere valore true o false
        void switchCellState(); //se è viva una cella puo diventare morta e viceversa
    }

    Griglia è un'entita composta da celle, in discussione con il committente si rileva che Griglia è un'entità a due dimensioni.
    Si rileva che una dimensione è la lunghezza, il numero di colonne, e una dimensione è la larghezza, numero di righe.
    La Grid deve poter individuare una cella specifica in base al numero di riga e di colonna

    Una non primitiva può essere costruita da altre primitive. Una primitiva è un operazione che non oposso realizzare senza consocere la rappresentazione concreta dell'entità, sono le uniche che conoscono come è fatta la struttura.

    public interface IGrid {
        public int getRowsNum(); // primitiva: per ottenere il numero di righe della grid
        public int getColsNum(); //primitiva: per ottenere il numero di colonne della grid
        public void setCellValue(int x, int y, boolean state); // non primitiva: per inserrie un valore in una specifica cella della grid
        public ICell getCell(int x, int y); //primitiva: per ottenre una cella specifica
        public boolean getCellValue(int x, int y); // per ottenere il valore di una cella specifica
        public void reset(); //non primitiva: si riferisce allo stato delle celle, pone tutte le celle allo stato morta.
    }

    Life è un'entità di cui il committente non conosce la struttura. è la logica del gioco Conway, contiene in se le regole del gioco.
    L'entità Life è composta da Grid, a sua volta composta. 
    public interface LifeInterface {
        void nextGeneration(); // Calcola l'evoluzione dello stato alla generazione successiva
        boolean isAlive(int row, int col); //Restituisce lo stato di una cella specifica
        void setCell(int row, int col, boolean alive); //Imposta lo stato di una cella

        // Restituisce il numero di righe e colonne
        int getRows();
        int getCols();

        ICell getCell(int x, int y); //Restituisce la Cella
        IGrid getGrid(); //Restituisce la grid
        void resetGrids(); //pulisce
        public String gridRep( ); // Restituisce una rappresentazione grafica testuale della grglia
    }

        life = Life.CreateLife(3, 3); // piccola griglia 3x3
    }

    @After
    public void down() {
        System.out.println("LifeTest | down");
    }

    @Test
    public void testCellInitialDead() {
        System.out.println("LifeTest | testCellInitialDead");
        // tutte le celle devono inizialmente essere morte
        for (int r = 0; r < 3; r++) {
            for (int c = 0; c < 3; c++) {
                assertTrue(!life.isAlive(r, c));
            }
        }
    }

    @Test
    public void testSetCellAlive() {
        System.out.println("LifeTest | testSetCellAlive");
        life.setCell(1, 1, true);
        assertTrue(life.isAlive(1, 1));
    }

    @Test
    public void testResetGrids() {
        System.out.println("LifeTest | testResetGrids");
        life.setCell(1, 1, true);
        life.resetGrids();
        assertTrue(!life.isAlive(1, 1));
    }
}
